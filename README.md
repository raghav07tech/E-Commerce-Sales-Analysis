# E-Commerce Sales Analytics Dashboard

## 📊 Project Overview
This project analyzes e-commerce sales data to identify revenue trends, top-performing categories, payment preferences, and customer behavior patterns. The analysis provides actionable business insights using Python data analysis and visualization.

## 📈 Key Findings

### Business Metrics
- **Total Revenue:** ₹757,278.08
- **Total Transactions:** 3,660
- **Total Customers:** 3,660
- **Average Transaction Value:** ₹206.91

### Top Performing Categories
1. **Clothing** - ₹115,314.84
2. **Books** - ₹111,149.35
3. **Home & Kitchen** - ₹110,328.08
4. **Sports** - ₹108,518.79
5. **Toys** - ₹107,289.69

### Payment Method Distribution
- Credit Card: 20.8% (₹156,018.02)
- UPI: 20.7% (₹154,684.81)
- Debit Card: 20.0% (₹153,362.26)
- Net Banking: 19.6% (₹146,804.40)
- Cash on Delivery: 19.0% (₹146,408.59)

### Important Insights
- **Highest Sales Month:** Month 10 (₹76,034.51)
- **Lowest Sales Month:** Month 11 (₹51,915.08)
- **Average Discount Offered:** 18.83%
- **Discount Range:** 0% to 50%
- **Highest Transaction:** ₹496.82
- **Lowest Transaction:** ₹5.89

## 📊 Analysis Methodology

### 1. Data Exploration
- Loaded and examined dataset structure
- Identified 3,660 transactions across multiple categories
- Verified data completeness (no missing values)

### 2. Data Cleaning
- Removed invalid records
- Standardized date formats
- Created derived columns (Purchase_Year, Purchase_Month, Purchase_Day)

### 3. Sales Analysis
- Calculated revenue by category, product, and payment method
- Identified monthly sales trends
- Analyzed discount impact on pricing

### 4. Visualization
- Created 6-chart dashboard showing:
  - Top categories by revenue
  - Monthly revenue trends
  - Payment method distribution
  - Discount impact analysis
  - Transaction amount distribution
  - Key business metrics summary

## Business Recommendations

1. **Clothing & Books** are revenue drivers - allocate more inventory
2. **Month 10 peak sales** - investigate what drove it and replicate the strategy
3. **Payment methods are balanced** - maintain support for all 5 options
4. **Discount strategy** - average 18.83% might be impacting margins; review pricing
5. **Customer retention** - focus on repeat purchases to increase average spending
